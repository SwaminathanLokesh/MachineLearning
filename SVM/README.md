# Support Vector Machines and Support Vector Regression

This project contains implementations of:

- Support Vector Machine (SVM)
- Support Vector Regression (SVR)

using Python and Scikit-learn.

---

# Topics Covered

## SVM

- Linear SVM
- RBF Kernel
- Classification
- Confusion Matrix
- Accuracy Score
- Spam Classification

## SVR

- California Housing Dataset
- Feature Scaling
- Regression
- MAE
- R2 Score
